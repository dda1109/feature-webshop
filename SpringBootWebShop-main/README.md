# SpringBootWebShop
Sử dụng Springboot làm web bán hàng
# Dự án có sử dụng:
 - Spring boot 2.6.8
 - JSP
 - SQL Server
 - Bootstrap 5
 - JPA
 - Spring mail
 - Session
 - admin page
# Dự án có chức năng:
 - Đăng nhập, đăng ký, quên mật khẩu
 - filter sản phẩm loại và giá
 - phân trang
 - giỏ hàng
 - Chi tiết sản phẩm
 - Quản lý sản phẩm, user
